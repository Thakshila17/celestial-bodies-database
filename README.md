# celestial-bodies-database
freeCodeCamp Relational Database Certification – Celestial Bodies Database project built with PostgreSQL.


This project was completed as part of the freeCodeCamp Relational Database Certification.

## Features
- PostgreSQL relational database
- Five tables: galaxy, star, planet, moon, and asteroid
- Primary keys and foreign key relationships
- Constraints including NOT NULL and UNIQUE
- Sample data inserted into all tables
- Database exported as `universe.sql`

## Technologies Used
- PostgreSQL
- SQL
